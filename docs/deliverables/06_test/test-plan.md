## 1. テスト方針

- **単体テスト**: Google Test + Google Mock を使用。実機不要（モックによる）。CI で自動実行。
- **結合テスト**: 実機（RPi 3B+ + MCP3008）上で `device-ctl` を起動し、対話メニューからチャネル読み出しを実行して動作確認。実機がない環境では結合テストの自動ジョブはスキップされる。
- **受け入れテスト**: 発注者立会いのもと手動実施し、本ドキュメント末尾のチェックリストで結果を記録・署名。
