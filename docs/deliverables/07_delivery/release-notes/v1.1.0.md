## 変更サマリ

### 新機能

- **`Sensor::read_raw_async()`**: 非同期チャネル読み出しAPI（CHG-003）  
  `std::thread` + コールバックで実装。UIスレッドをブロックしない。

- **`device-ctl --async`**: CLIに非同期モードを追加（CHG-004）  
  `device-ctl --async` で起動し、対話メニューからチャネルを非同期読み出し可能。

### バグ修正

- **EAGAIN リトライ（CHG-002）**: SPI転送でEAGAINが返る場合に最大3回リトライするように修正。  
  高負荷環境での転送失敗率がほぼ0になった（実測: 100回中0件）。

### インフラ

- GitHub Actions による CI パイプラインを導入（ビルド・テスト・cppcheck・カバレッジ・Artifacts）

## アップグレード手順

```bash
# 旧バイナリのバックアップ
cp libsensor.so.1.0.0 libsensor.so.1.0.0.bak

# 新バイナリの配置
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release
cmake --build build
sudo cmake --install build

# バージョン確認
device-ctl --version
```

## 後方互換性

- `libsensor.so.1` の ABI 変更なし（`read_raw_async` は**追加**のみ）
- `spi-hal/v1.0.0` との組み合わせは**非推奨**（EAGAIN未対応のため `spi-hal/v1.0.1` を使用すること）
