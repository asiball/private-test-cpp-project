## 1. 物理インターフェース

| 信号名 | 方向 | ピン（Raspi3B+） | 説明 |
|---|---|---|---|
| SCLK | Master → Slave | GPIO11 (Pin 23) | クロック |
| MOSI | Master → Slave | GPIO10 (Pin 19) | マスタ送信 |
| MISO | Slave → Master | GPIO9  (Pin 21) | マスタ受信 |
| CE0  | Master → Slave | GPIO8  (Pin 24) | チップセレクト（アクティブLow） |

## 2. 電気的仕様

| 項目 | 仕様 |
|---|---|
| 動作電圧 | 3.3V |
| 最大クロック | 2 MHz（CHG-001） |
| データフォーマット | 8ビット、MSBファースト |
| SPIモード | MODE 0（CPOL=0, CPHA=0） |

## 4. レジスタアドレスマップ（参考）

| アドレス | アクセス | 説明 |
|---|---|---|
| 0x00 | R | デバイスID（固定値: 0xA5） |
| 0x01 | R | ステータスレジスタ |
| 0x10 | R/W | 設定レジスタ1 |
| 0x11 | R/W | 設定レジスタ2 |
| 0x20 | R | 測定データ（4バイト） |

> アドレスビット7（MSB）が1の場合は読み出し要求とみなす（SPIデバイス仕様による）。

## 5. ioctl コマンド一覧

| コマンド | 用途 |
|---|---|
| `SPI_IOC_WR_MODE` | SPIモード設定 |
| `SPI_IOC_WR_BITS_PER_WORD` | ビット幅設定 |
| `SPI_IOC_WR_MAX_SPEED_HZ` | クロック速度設定 |
| `SPI_IOC_MESSAGE(n)` | n個のspi_ioc_transferを1トランザクションで送受信 |
