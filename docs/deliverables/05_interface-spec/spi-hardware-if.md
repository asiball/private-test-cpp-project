## 1. 物理インターフェース

| 信号名 | 方向 | RPi 3B+ ピン | MCP3008 ピン | 説明 |
|---|---|---|---|---|
| SCLK | Master → Slave | GPIO11 (Pin 23) | CLK (13) | クロック |
| MOSI | Master → Slave | GPIO10 (Pin 19) | DIN (11) | マスタ送信（コマンド） |
| MISO | Slave → Master | GPIO9  (Pin 21) | DOUT (12) | マスタ受信（変換結果） |
| CE0  | Master → Slave | GPIO8  (Pin 24) | CS/SHDN (10) | チップセレクト（アクティブLow） |

> MCP3008 のアナログ入力 CH0〜CH7（ピン1〜8）に測定対象を接続する。
> VDD/VREF は 3.3V、AGND/DGND は GND に接続する。

## 2. 電気的仕様

| 項目 | 仕様 |
|---|---|
| 動作電圧 | 3.3V |
| 最大クロック | 2 MHz（CHG-001） |
| データフォーマット | 8ビット、MSBファースト |
| SPIモード | MODE 0（CPOL=0, CPHA=0） |

## 4. MCP3008 コマンド／データフォーマット

MCP3008 はレジスタを持たず、1 回の A/D 変換を **3 バイトのフルデュプレクス転送**で完結する。

### 4.1 送信（MOSI）

| バイト | 値 | 説明 |
|---|---|---|
| TX[0] | `0x01` | スタートビット |
| TX[1] | `0x80 \| (ch << 4)` | SGL/DIFF=1（シングルエンド）+ チャネル選択 D2..D0 |
| TX[2] | `0x00` | 変換クロック供給用ダミー |

### 4.2 受信（MISO）

| バイト | 説明 |
|---|---|
| RX[0] | 無効（ドントケア） |
| RX[1] | 下位 2bit が変換結果の bit9..bit8 |
| RX[2] | 変換結果の bit7..bit0 |

10bit の変換結果と電圧は次式で復元する:

```
raw     = ((RX[1] & 0x03) << 8) | RX[2]   // 0〜1023
voltage = raw * Vref / 1023               // [V]
```

## 5. ioctl コマンド一覧

| コマンド | 用途 |
|---|---|
| `SPI_IOC_WR_MODE` | SPIモード設定 |
| `SPI_IOC_WR_BITS_PER_WORD` | ビット幅設定 |
| `SPI_IOC_WR_MAX_SPEED_HZ` | クロック速度設定 |
| `SPI_IOC_MESSAGE(n)` | n個のspi_ioc_transferを1トランザクションで送受信 |
